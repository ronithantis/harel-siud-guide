# 🤝 מדריך להגשת תביעת סיעוד בהראל

כלי חינמי שמלווה בני משפחה שלב אחר שלב בתהליך הגשת תביעת סיעוד בהראל ביטוח.

## ✨ תכונות
- מדריך שלב אחר שלב ב-9 שלבים פשוטים
- תמיכה בתביעות תפקודיות, קוגניטיביות ומשולבות
- צ'קליסט מסמכים עם מעקב התקדמות
- הפקת דוח אישי להדפסה
- שמירת טיוטה אוטומטית בדפדפן
- הפניה לעו"ד מתמחה (אופציונלי)
- סרטון הסבר מובנה
- עיצוב RTL מלא בעברית
- מותאם לנייד ולמחשב

## 🔐 פרטיות
כל המידע נשמר בדפדפן בלבד (localStorage).
אין שרת, אין מסד נתונים, אין איסוף מידע.
הפניה לעו"ד כוללת הסכמה מדעת לפי תיקון 13 לחוק הגנת הפרטיות.
