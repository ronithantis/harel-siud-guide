# 🤝 מדריך להגשת תביעת סיעוד בהראל

כלי חינמי שמלווה בני משפחה שלב אחר שלב בהגשת תביעת סיעוד בהראל ביטוח.

## למה הכלי הזה קיים?

טופס תביעת הסיעוד של הראל הוא מסמך של 21 עמודים, מלא בשפה משפטית ורפואית. בני משפחה שמנסים להגיש תביעה עבור ההורים שלהם נתקלים בתהליך מסורבל ומבלבל.

הכלי הזה **לא ממלא את הטופס** עבורכם — הוא **מלווה אתכם** בתהליך:
- מסביר כל שלב בשפה פשוטה
- עוזר לרכז את כל הפרטים והמסמכים הדרושים
- מפיק דוח אישי שמראה מה מוכן ומה עוד חסר
- מבדיל בין מה שאתם ממלאים לבין מה שאחרים (רופא, עו"ד, מטפל) צריכים למלא

## תכונות

- ✅ אשף מודרך ב-9 שלבים
- 📋 צ'קליסט מסמכים אינטראקטיבי
- 🧠 הסברים מותאמים לתביעות תשישות נפש (דמנציה/אלצהיימר)
- 🦽 הסברים על מוגבלות תפקודית ו-6 פעולות ADL
- 📄 הפקת דוח מוכנות להדפסה
- 🔒 כל המידע נשמר בדפדפן בלבד — אינו נשלח לשום שרת
- 📱 מותאם למובייל ודסקטופ
- 🇮🇱 עברית מלאה, ממשק RTL

## שימוש

הכלי הוא קובץ HTML יחיד. אפשר:

1. **לפתוח ישירות** — הורידו את `index.html` ופתחו בדפדפן
2. **לגלוש באתר** — [הלינק של GitHub Pages יופיע כאן אחרי ההפעלה]

## הפעלת GitHub Pages

1. העלו את הקבצים ל-repository ב-GitHub
2. לכו ל-**Settings** → **Pages**
3. תחת **Source** בחרו **Deploy from a branch**
4. תחת **Branch** בחרו **main** ותיקיית **/ (root)**
5. לחצו **Save**
6. תוך דקה-שתיים תקבלו לינק: `https://YOUR-USERNAME.github.io/REPO-NAME`
